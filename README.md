# Trie Autocomplete System 

## Description:
This project is a prefix based trie transversal for related word retrievals when the user types something maybe in the search or somewehere else. I built the project to implement my new found knowledge on the trie data structure that i gained from the Princetons Algorithm Course online. The code can be run from any IDE that supports the Python Programming Language. Its a simple code with some logic for retrieving the related words based on what the user has typed. The project has not used any extra libraries and has only showcased Python's OOP concept.

## Features:
- Insert words
- Search full words
- Autocomplete using DFS
- Case-insensitive search
